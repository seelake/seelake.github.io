Total VST compatibility!

AudioStation has been tested with thousands of freeware and commercial VST plugins. Tests results confirmed the real compatibility with every 64-bit VST plugin running on Windows 7. Due to the integration of a 32-bit to 64-bit bridge, the 32-bit plugins can be also used, including older versions running on Windows XP.

Plugin installation

Unlike other similar devices, AudioStation doesn't require special versions of plugins. Plugins get installed with the normal procedures expected from developers, with no need for tricky operations and/or to compile specific support files.

Copy protection

To ensure using any VST plugins, beyond the full VST protocol compatibility, AudioStation supports any copy protection system: iLok, eLicenser (formerly SyncroSoft), Challenge-Response and Serial Number.
