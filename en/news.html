March 1, 2015

Jack Audio Server for AudioStation

To overcome the limitations of the ASIO driver - which DO NOT allows you to use the audio card with two or more ASIO applications simultaneously - AudioStation has been equipped with Jack Audio Server.
With this capability you can use the AudioStation VST-host together with other software applications, such as Ableton Live, Pro Tools, Cubase, Nuendo, Sonar, Wavelab and many others, sharing the audio I/Os of AudioStation between applications.
Experienced users will also use JackControl (the Jack Audio Server's audio patchbay) to freely route the audio signals between applications.

January 2, 2013

New for 2013: the biggest upgrade we've ever released!

Happy new year!

Our starts brilliantly, since after 6 months of hard work we are glad to announce a new version of the AudioStation's software, with lots of interesting new features: multi-in and multi-out, new preload functions and support for USB-MIDI controllers are just some of them.

Here they are:

  • Multi-In: it is useful to process audio signals with multichannel plugins (eg. Surround Sound 5.1, 7.1 and similar). With the Multi-In option it is possible to set the audio channels with 6, 8 or 10 audio inputs an equal number of outputs.
  • Multiout: it is useful to route to separated outputs signals from multitymbral plug-ins with multiple outputs, for example to process and/or to record separately signals from a drum or others.
  • Performance Preload: allow to sets the preload of the Performance and related Patches with a single click.
  • Automatic Preload of Next Performance: it is a global setting for automatically preload the next Performance (and all of its Patches), without additional settings.
  • Preload on the fly...: while a set up is in use simply indicate a new Performance and preload will start automatically.
  • Support for USB-MIDI controllers: allows to connect to AudioStation any MIDI controller-USB or Ethernet.
  • MIDI bypass of the VST insert slots: the On/Off status of the VST Insert slots now can be controlled also via MIDI.
  • AND function for the MIDI control of the plugin's parameters: allows to define a complex set of changes for adlusting the parameters of plugins via MIDI.
  • Toggle function for the MIDI control of the plugin's parameters: it changes the On/off behaviour of buttons, foot switches and notes from temporary to permanent, allowing to perform On status or Off status alternately each time they are pressed.
  • Note Toggle: it changes the behaviour On/Off of the keys from temporary to permanent, allowing to perform Note On or Note Off alternately each time the keys are pressed.
  • Linear Knob: allows to adjust the knobs by dragging the mouse vertically (instead of the circular motion).
  • Tempo adjustment with LiveControl: the Tempo now can be adjusted even with LiveControl.
  • Editable plugins list: the list of plugins is now contained in a file easily editable.

New features introduced with the previous update:

  • VST3 Support: allows to use even the new VST3 plugins.
  • Save All: it is useful to save the Performance and every channel Patch with a single command and with the same name and memory number..
  • Continuous MIDI Clock: allows to send MIDI Clock signal without having to start the playback of audio player.
