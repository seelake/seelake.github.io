Controllo con iPad (Wi-Fi)

In aggiunta all’uso di Live Control, monitor, tastiera e mouse, AudioStation può essere controllata via Wi-Fi con un iPad. Per consentire di operare ovunque anche in assenza di rete Internet, il segnale Wi-Fi viene generato da AudioStation senza bisogno di un computer o altro hardware aggiuntivo.
Grazie al supporto completo delle gestualità iOS, conotrollare le funzionalità del VST Host /Mixer di AudioStation e regolare i parametri dei plug-in è davvero facile.

 

In aggiunta, è possibile visualizzare nel iPad una versione virtuale del remote controller di AudioStation, che può essere utilizzato in alternativa o simultaneamente alla versione reale.

 

Oltre al controllo del VST Host /Mixer, è anche possibile controllare tutte le altre funzionalità dell’apparecchio e di qualsiasi software utilizzato.

Controllo via Ethernet

AudioStation può essere controllata anche via Ethernet da computer Windows e Mac.
